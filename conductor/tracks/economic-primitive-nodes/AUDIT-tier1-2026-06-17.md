**Confirmed chain-free.** None of the Tier-1 handlers injects `INftManager`,
`ISwapManager`, `IBlockchainProvider`, `IBlockchainOperationManager`, or any
provider/DEX abstraction. The only injected manager across the `Holon*` set is
`IHolonManager` (a metadata/graph store manager — no chain), and the three
control/transform nodes (`ComposeOutputs`, `GateCheck`, `Emit`) inject nothing
at all. Every one of them leaves `RequiresChainCapability` at the SPI default
`false` — verified mechanically by `ChainCapabilityFlagInvariantTests` (T11),
which reflects over the registered handler set rather than a hand-maintained
list, so the invariant cannot drift.

> Note: `IHolonManager` is the holon metadata/graph store boundary. It is
> chain-free by construction — the chain seam lives in `INftManager` /
> `ISwapManager` / `IBlockchainProvider`, none of which any Tier-1 handler
> touches.

## D9 verdict — **NO new `Transform`/`Branch` node**

The existing Tier-1 set already covers every generic shaping need, so no extra
node is added:

- **Metadata / value / graph transform** → `HolonInteract` (reparent, peer
  edits, metadata mutation) covers in-place shaping; `HolonClone` /
  `HolonMoveSubtree` / `HolonPropagate` cover structural transforms.
- **Branching** → `GateCheck`. A `Fail` result propagates through the engine's
  existing failed-predecessor skip (`QuestManager.cs:262-289`), gating the
  downstream path — `GateCheck` *is* the branch primitive; no separate `Branch`
  node carries logic the engine doesn't already express.
- **Rollup / value composition** → `ComposeOutputs` gathers upstream outputs;
  `HolonCompose` produces a subtree rollup view.

No "pure value-shaping transform that is NOT a holon edit" gap was found that
these three do not already serve. Per D9, the default holds: **add no extra
node.** If a future tenant flow needs a non-holon scalar transform that
`ComposeOutputs` + `GateCheck` cannot express, revisit and add a Tier-1
`Transform` node (`RequiresChainCapability == false`) at that point — documented
fallback, not today's default.

## Outcome

- Tier-1 set is coherent and **chain-free** (all `RequiresChainCapability ==
  false`), mechanically enforced by the T11 invariant test.
- **D9 = no new node.** `HolonInteract` (transform), `GateCheck` (branch), and
  `ComposeOutputs` (rollup) close the generic-transform surface.
